## 📖 Overview

Data Detective AI is an autonomous multi-agent data science platform that automatically profiles datasets, detects anomalies, discovers patterns, generates AI-powered insights, recommends machine learning models, and creates professional business reports.

The platform eliminates manual exploratory analysis and helps analysts, researchers, and businesses derive meaningful insights within minutes.

## ✨ Features

### 📊 Data Profiling Agent
- Dataset health scoring
- Missing value detection
- Duplicate detection
- Data type inference
- Memory usage analysis

### 🔬 Analytics Agent
- Outlier detection
- Correlation analysis
- Feature importance analysis
- Classification/Regression auto-detection

### 🧠 AI Insight Agent
- Gemini-powered business insights
- Executive summaries
- Data quality recommendations
- Action plans

### 🤖 Recommendation Agent
- ML model recommendations
- Preprocessing suggestions
- Evaluation metric recommendations

### 📑 Report Agent
- Professional PDF report generation
- Business-ready insights
- Automated documentation
